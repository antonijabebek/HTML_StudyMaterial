Grafika u HTML-u

Sliku unutar paragrafa možemo umetnuti s elementom img. Ovaj element nema završnu oznaku. Pomoću atributa src dodjeljujemo izvor slike. Atribut alt ima vrijednost koja će biti prikazana ukoliko se slika ne bude mogla učitati. Atributi width i height određuju širinu i visinu slike unutar web stranice

U HTML možemo umetnuti i SVG (Scalable Vector Graphics) pomoću elementa svg
